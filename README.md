# 42_Born2beroot

## About the project
  
This project is a part of Milestone 1 of Common Core at 42 school. It aims to teach more on virtualisation and server administration.

### Before Delivering

Make sure you understand what your code is doing
- Follow subject guidelines on how to deliver the project
- Do not open your virtual machine, otherwise ID will change
  - In case it happens during evaluation, ask the previous evaluator to explain what occurred to your next evaluator
- Can you answer all the questions in the guides provided?
